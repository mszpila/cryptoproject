## Installation

```bash
$ yarn install
```

Prepare YAML configs based on examples.

## Running the app

```bash
# development
$ yarn start

# development watch mode
$ yarn start:dev

# development watch mode with external services
$ yarn start:ext

# production watch mode
$ yarn start:prod

# production mode
$ yarn start:prod
```

## Test

```bash
# in-memory tests
$ yarn test

# external tests
$ yarn test:ext
```

## Technology stack

- Nest framework
- MongoDB
